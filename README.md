<a href="https://sambuxc.github.io/grid-selector/" target="_blank" alt="Grid selector component using domql and parceljs">View Demo</a>

### Setup

1. Clone the repo
```
git clone git@https://github.com/Sambuxc/grid-selector.git
```

2. Install scripts
```
yarn
```

3. Run the project
```
yarn start
```
